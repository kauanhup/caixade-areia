# Projeto Caixa de Areia Interativa - Rio Branco (MT)

## 📖 Sobre o Projeto

Este projeto utiliza como base o **[Augmented Reality Sandbox (AR Sandbox)](https://arsandbox.ucdavis.edu)**, desenvolvido pelo Centro W.M. Keck para Visualização Ativa em Ciências da Terra da UC Davis (KeckCAVES), com apoio da National Science Foundation (Grant No. DRL 1114663).  

Mais informações podem ser encontradas em nosso blog: [clique aqui!](https://caixade-areia.com)

## 🚀 Instalação Simplificada

O processo agora é totalmente automatizado:

1. **Baixe o instalador**  
   [instalador.sh](https://github.com/kauanhup/caixade-areia/raw/main/assent/instalador.sh)

2. **Execute o instalador**  
   - Clique duas vezes no arquivo baixado.  
   - Digite sua senha quando solicitado.  
   - O script instalará todos os componentes necessários (VRUI, Kinect, SARndbox e customizações do projeto).  

3. **Abrir o programa**  
   - O atalho **Caixa-de-areia.sh** será criado na área de trabalho.  
   - Clique duas vezes para iniciar o projeto.
